# Prompt — 35-dashboard-builder

The original prompt used to generate this motion graphic:

Duration: 25 seconds

An analytics product open in a browser frame. A blank dashboard canvas — white, gridded faintly, empty. A "Add widget" button in the top right. A cursor clicks it. A widget picker panel slides in from the right — a scrollable list of widget types: Line Chart, Bar Chart, Metric Card, Table, Pie Chart, Funnel. The cursor clicks Line Chart. The panel closes. A line chart widget appears on the blank canvas, centered — a default chart with placeholder data, draggable resize handles at its corners.
The cursor drags the chart to the top left quadrant of the canvas. Clicks "Add widget" again. Selects Metric Card. A metric card appears — the cursor drags it to the top right. Clicks Add Widget again. Bar Chart. Drags it to the bottom left. One more — Table. Bottom right. The dashboard now has four widgets in a 2×2 grid, all showing placeholder data.
The cursor clicks the line chart. A configuration panel opens on the right — data source selector, metric selector, date range. The cursor selects a data source from a dropdown. The chart updates — real data flowing in, the line drawing itself left to right across the chart area. The cursor closes the config panel. Clicks the metric card. Selects a metric. The placeholder number counts up to a real value. Closes. Clicks the bar chart. Configures it. Real bars draw in. Closes. The table auto-populates when clicked — rows appearing one at a time, fast, filling the widget. The cursor clicks away. The dashboard is complete — four widgets, all live, all real data. The cursor moves to a "Share" button in the top nav and clicks it. A shareable link copies to clipboard. A toast confirms it. Done.
