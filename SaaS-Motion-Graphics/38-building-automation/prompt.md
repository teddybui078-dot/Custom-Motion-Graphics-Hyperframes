# Prompt — 38-building-automation

The original prompt used to generate this motion graphic:

Building an Automation
Duration: 22 seconds

A workflow automation tool open in a browser frame. A blank canvas with a faint grid — the flow builder. A cursor clicks "Add Trigger" in the top left. A panel slides in from the left — a list of trigger types with icons. The cursor scrolls briefly and clicks "New Form Submission." A trigger block appears on the canvas — a rounded rectangle, teal, labeled with the trigger name.
The cursor clicks the "+" below the trigger block. An action picker appears — the cursor selects "Send Slack Message." A Slack action block appears below the trigger, connected by a thin animated line, a traveling dot moving down it continuously showing flow direction. The cursor clicks the Slack block — a config panel opens on the right. A channel is selected from a dropdown. A message template types itself in, with dynamic variables highlighted in blue — {submitter name}, {form title}. The panel closes.
The cursor adds two more action blocks — "Create Trello Card" and "Send Email Confirmation" — each connecting below the previous with the same animated line. The automation is four blocks deep. The cursor clicks Test Automation in the top right. Each block lights up green in sequence, top to bottom, as the test runs — a satisfying cascade of confirmation. All four green. The cursor clicks Activate. A status badge in the top left switches from Draft to Live. The traveling dots on the connecting lines speed up slightly — the automation is running.
