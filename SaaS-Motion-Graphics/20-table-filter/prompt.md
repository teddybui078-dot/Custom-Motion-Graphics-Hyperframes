# Prompt — 20-table-filter

The original prompt used to generate this motion graphic:

A data table fills the frame — dozens of rows, multiple columns, dense with information. A search bar sits above it. A cursor clicks into it. A query types itself in — three words, deliberate pace. As the third character appears, the table begins filtering in real time — rows that don't match fade out and collapse, the table contracting smoothly as they disappear. By the time the query is complete, eight rows remain out of dozens. They sit in the table with generous spacing, easy to read. The cursor then clicks a filter chip to the right of the search bar — a dropdown opens, a category is selected, the dropdown closes. Three more rows disappear. Five rows remain. The cursor moves away from the search bar. The filtered table holds — clean, precise, exactly what was asked for.
