# Prompt — 04-node-flow

The original prompt used to generate this motion graphic:

this now using hyperframes: Five minimalist nodes sit in a horizontal line on #F5F5F7. Each is a white rounded square with a single stroke icon inside. They are invisible at the start.
The nodes spring into existence one at a time, left to right. Each node pops with a fast spring — it slightly overshoots scale before settling. As each node lands, its label materializes directly below it with a soft fade up, no delay after the pop.
Once all five nodes are placed, a thin connector line grows from the right edge of node 1 toward node 2 — it draws itself steadily, left to right, like ink being laid down. The instant the line fully connects, a small glowing dot launches from node 1 and travels the line at a brisk, even pace, arriving at node 2 just as the viewer registers the connection. The dot doesn't bounce or ease — it travels clean and disappears on arrival.
This exact sequence — line draws, dot travels — repeats for each subsequent connection, one at a time, unhurried. When the final dot reaches node 5, all five nodes simultaneously emit a single expanding ring from their centers. The ring grows outward and fades before it doubles the node size. After that, absolute stillness.
