# Prompt — 37-filter-crm

The original prompt used to generate this motion graphic:

Filtering a CRM
Duration: 18 seconds

A CRM contacts list fills the frame — hundreds of rows, each a contact with name, company, deal size, stage, last contacted date, owner. Dense with data. A cursor clicks a Filter button above the table. A filter bar slides down from below the table header — three filter dropdowns appearing: Field, Condition, Value.
The cursor configures the first filter: Field → Deal Size, Condition → Greater than, Value → $10,000. The table filters instantly — rows that don't match collapse smoothly, the table contracting as they disappear. 340 contacts become 89.
The cursor clicks "Add Filter." A second filter row appears. Field → Stage, Condition → Is, Value → Negotiation. The table filters again — 89 becomes 23. The cursor adds one more filter: Owner → Is → Unassigned. 23 becomes 7. Seven high-value deals in negotiation with no owner. The cursor selects all seven rows — checkboxes filling one by one down the filtered list. Clicks "Assign Owner" in the bulk action bar that appears. Selects a name from a dropdown. All seven rows update simultaneously — the Owner column populating across all selected rows in a single frame. The filter clears. The full table returns. Seven rows now have an owner that didn't before.
