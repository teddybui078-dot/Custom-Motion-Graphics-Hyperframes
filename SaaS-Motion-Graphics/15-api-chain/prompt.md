# Prompt — 15-api-chain

The original prompt used to generate this motion graphic:

Six nodes arranged in a horizontal line on a white canvas. Each node is a rounded rectangle — wider than it is tall, just enough room for a short label inside. They are all invisible at the start.
They appear one at a time left to right, each springing in with a single overshoot pop. As each node lands its label fades in inside it — not before, not after, exactly on the same frame the node settles. The labels are short API method names. POST. GET. AUTH. PARSE. CACHE. RETURN.
Once all six are placed, a directional arrow line draws from the right edge of node 1 toward node 2 — the arrowhead travels with the line tip, arriving at the node edge at the same moment the line finishes drawing. The arrow is the motion. The line is just its trail.
This repeats node to node, one connection at a time. Each arrow takes the same duration to draw regardless of the gap between nodes — the speed adjusts to the distance. When the final arrow lands on RETURN, the node flashes green for 4 frames — a hard color cut, then back to white. A small 200 OK label materializes above it in green monospace. The chain holds.
