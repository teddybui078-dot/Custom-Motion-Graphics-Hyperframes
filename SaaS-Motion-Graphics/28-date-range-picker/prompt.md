# Prompt — 28-date-range-picker

The original prompt used to generate this motion graphic:

Date Range Picker
Duration: 12 seconds

An analytics dashboard on white. A date range selector sits in the top right — currently showing "Last 30 days." A cursor clicks it. A calendar dropdown appears — two months side by side. The cursor clicks a start date on the left calendar — the date highlights blue. The cursor drags across to an end date on the right calendar — the range fills in blue as the cursor moves, each date cell in the range highlighting as the cursor passes over it. The cursor releases on the end date. The dropdown closes. The dashboard transitions — all charts and metrics updating simultaneously with a smooth 20-frame crossfade to the new data range. The date selector now shows the custom range. The data tells a different story for this period.
