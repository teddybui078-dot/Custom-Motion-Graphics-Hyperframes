# Prompt — 05-decision-tree

The original prompt used to generate this motion graphic:

ok what about this now:

A single dark pill-shaped node appears at the top center of a white canvas. It fades in with a subtle scale-up from slightly small — not a spring, just a clean ease-out settle. It feels like a card being placed on a table.
Two curved bezier lines grow downward from its bottom edge simultaneously, arcing left and right toward two mid-tier nodes. The lines draw at the same pace, perfectly mirrored. As each line tip arrives at its destination, the mid node materializes with a spring pop — identical to the root node but lighter in color.
The same thing happens again: four lines branch outward from the two mid nodes simultaneously, drawing at the same pace. The four leaf nodes all pop at the same moment.
Once all nodes are placed, the label text inside each leaf node types itself on — one character at a time, fast enough to feel live but slow enough to read. A blinking text cursor sits at the end of each label as it types. Once fully typed, the cursor blinks three more times and disappears. All four labels type in parallel, each starting at the same moment.
