# Prompt — 33-running-a-search

The original prompt used to generate this motion graphic:

Running a Search
Duration: 15 seconds

A fully populated product interface on white — a document management tool. Hundreds of files visible in a grid, all labeled, all dated. A cursor clicks a search bar at the top of the screen. It expands — growing wider, a blinking cursor inside it, the rest of the interface dimming slightly behind it. A search query types itself in — three words, deliberate. As the second word is typed, results begin appearing below the search bar in a dropdown — live results, updating with each character. The results show file names, file types, last edited dates, which team member last touched each one.
The cursor stops typing. Six results are visible in the dropdown. The cursor clicks the third result. The dropdown closes. The file opens — a full document view, the search terms highlighted in yellow wherever they appear in the document. The cursor clicks the first highlight. The document scrolls to that section. The cursor clicks the second highlight. Scrolls again. The user found exactly what they needed in under 8 seconds of screen time. The document holds open.
