# Prompt — 44-generating-report

The original prompt used to generate this motion graphic:

Generating a Report
Duration: 20 seconds

An analytics tool on white. A "Reports" section in the sidebar — the cursor clicks it. A reports library appears: a grid of report templates, each showing a thumbnail preview and a name. The cursor hovers over several — tooltips appear with descriptions. The cursor clicks "Monthly Performance Report." A configuration screen opens — date range picker, metric selector, a team filter, a format dropdown. The cursor sets the date range to last month. Selects six metrics by clicking their checkboxes one by one. Leaves the format as PDF.
Clicks Generate. A progress indicator appears — not a spinner, but a step-by-step breakdown: Fetching data → green checkmark. Processing metrics → green checkmark. Building layout → green checkmark. Rendering PDF → progress bar filling. The PDF renders in 3 seconds. A preview appears — a properly formatted multi-page report, branded, with charts and tables visible in the preview pane. The cursor clicks Download. A download toast appears. The cursor clicks Share — a modal opens, an email address types in, Send clicked. The report has been generated, downloaded, and shared in under 8 seconds of configuration time.
