# Prompt — 22-collab-cursors

The original prompt used to generate this motion graphic:

A document editor fills the frame — a clean writing interface, a long block of text. A single cursor blinks at a paragraph. Then a second cursor appears — a different color, labeled with a name tag floating above it. It moves independently, scrolling to a different section of the document, clicking into a heading. A third cursor appears, labeled differently, also a different color. It selects a sentence — the selection highlight appears in its color. The three cursors move simultaneously but independently — each doing something different, each in a different part of the document. A small avatar stack appears in the top right of the editor showing three profile icons. The document is being worked on by three people at once. None of the cursors are aware of the others. The document holds all of them.
