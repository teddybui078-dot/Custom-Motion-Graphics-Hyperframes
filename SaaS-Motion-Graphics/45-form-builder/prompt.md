# Prompt — 45-form-builder

The original prompt used to generate this motion graphic:

Setting Up a Form
Duration: 22 seconds

A form builder tool on white. A blank form canvas on the left, a field library panel on the right. The form has a title field at the top already — "Untitled Form" in editable gray text. A cursor clicks it and types a real form name.
The cursor drags a Text field from the right panel onto the canvas. It drops in with a spring bounce — a labeled input field appearing on the form. The cursor clicks its label and types a question. Drags a second field — Dropdown. Drops it below the first. Clicks its label, types. Clicks the dropdown options area — an inline editor opens, four options type themselves in one at a time. The cursor drags a File Upload field. Then a Date Picker. The form now has four fields, all labeled, all configured.
The cursor clicks a Settings gear icon. A panel opens — toggle switches for Required fields, Email notification on submission, Redirect URL after submit. Three toggles switched on. The cursor clicks Preview in the top right. The form renders as it would appear to a respondent — clean, white, properly spaced, the fields exactly as configured. The cursor fills in the preview form, clicking through each field. Clicks Submit. A success message appears on the preview. The cursor clicks back to Editor. The form is ready to publish.
