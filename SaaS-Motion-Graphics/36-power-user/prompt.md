# Prompt — 36-power-user

The original prompt used to generate this motion graphic:

Duration: 30 seconds

A product interface fills the frame. A cursor enters — and moves with the speed and precision of someone who knows exactly what they're doing. No hesitation. No searching for buttons. Pure muscle memory.
First 8 seconds: The cursor opens a command palette with a keyboard shortcut — a search bar appearing over the interface. Types three characters. A command appears. Enter. A complex automation is triggered — tasks reassigning themselves, a report generating, a Slack notification firing, all visible on screen simultaneously as the automation runs. The cursor hasn't clicked a single button.
Next 8 seconds: The cursor opens a data view, applies four filters in rapid succession using keyboard shortcuts, exports the result with two keystrokes, and shares the export link before the export animation has finished playing. Speed without error.
Next 8 seconds: The cursor builds a new automation from scratch — dragging trigger and action blocks into a flow builder, connecting them with lines, setting conditions, clicking Test. The test runs — a live preview of the automation executing, each step lighting up green in sequence. The cursor clicks Activate. The automation is live.
Final 6 seconds: The cursor stops. The interface holds — automations running in the background, the data view filtered and exported, the command palette closed. The product is doing work without the cursor touching anything. A headline fades in: "The last tool you'll need to learn." Product name. CTA. Cut to white.
