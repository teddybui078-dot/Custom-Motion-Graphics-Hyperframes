# Prompt — 23-dashboard-tour

The original prompt used to generate this motion graphic:

White canvas. A product UI — a dashboard, fully detailed — sits in a browser frame. The cursor enters from the left edge. It moves to a chart and hovers — a tooltip appears with data. It moves to a metric card and hovers — the card brightens slightly. It moves to a filter dropdown and clicks — options appear. It selects one. The chart updates — the data redraws, bars shifting to new values, a smooth 20-frame transition. The cursor moves to a date range picker and clicks — a calendar appears. It selects a range. The chart updates again. The cursor moves away and exits the right edge of the frame. The dashboard holds with the new data applied. The cursor was the narrator. The product did the rest.

duration=14secs
