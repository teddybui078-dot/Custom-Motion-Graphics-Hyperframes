# Prompt — 40-product-character

The original prompt used to generate this motion graphic:

The Product as a Character
Duration: 20 seconds

White canvas. The product's logo mark — just the icon, not the wordmark — sits centered, small, alone. It is still. Then it blinks — two small curved lines appear briefly above it suggesting eyes, then disappear. The logo is alive.
It begins moving — bouncing gently, exploring the canvas, leaving a faint trail behind it like a comet tail. It moves to the left edge, finds it, bounces back. Moves upward, bounces off the top. It is learning the space it inhabits. As it moves, small UI elements fall from above — a button, a chart, a data card, a notification bell — each landing on the canvas with a spring bounce. The logo moves toward each element and absorbs it — scaling up slightly each time, the absorbed element disappearing inside it. After absorbing five elements the logo is noticeably larger.
It bounces back to center, now bigger. It sits still. A tagline fades in around it: "[Product]. Everything in one place." The logo pulses once — a happy pulse, if a logo can look happy. Holds.
