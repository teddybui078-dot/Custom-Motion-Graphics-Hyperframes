# Prompt — 25-file-preview

The original prompt used to generate this motion graphic:

File Preview
Duration: 12 seconds

A file grid on white. A cursor double-clicks a document card. The card expands — not opening a new window, but scaling outward from its position in the grid, growing to fill the center of the frame as a preview modal. The surrounding grid dims to 60% opacity behind it. Inside the modal, the document renders — pages loading in from top to bottom, a subtle shimmer passing over each page as it resolves from skeleton to content. The cursor scrolls — the document pages travel upward inside the modal, smooth and physical. The cursor clicks a close button. The modal compresses back down to its original card position in the grid, the grid brightening back to full opacity. The card is exactly where it was.
