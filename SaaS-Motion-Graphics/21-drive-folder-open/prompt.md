# Prompt — 21-drive-folder-open

The original prompt used to generate this motion graphic:

A browser frame fades in on white. Inside it, Google Drive's home screen — the familiar grid of recent files, the left sidebar with My Drive, Shared, Starred. Everything is static and fully populated. A cursor enters and hovers over a folder — the folder card lifts slightly, its shadow deepening, a subtle scale increase. The cursor clicks. The folder opens — the grid of files inside fades in with a staggered cascade, each file card appearing 4 frames after the previous, left to right, top to bottom. The folder is full. The cursor moves away. The contents breathe.
